Feedstock for [oneloop](https://bitbucket.org/hameren/oneloop) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

OneLOop: fast numerical evaluation of one-loop scalar integrals

## Install

```bash
conda install -c hep-forge -c conda-forge oneloop
```
